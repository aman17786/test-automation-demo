# Test Automation Demo

## Overview
This repository demonstrates GitHub MCP integration with AI-powered testing platform.

## MCP Servers Integrated
- MySQL - Database operations and validation
- Azure DevOps - CI/CD pipeline integration
- Playwright - Browser automation
- Sauce Labs - Mobile device testing
- Excel - Test reporting
- REST API - API testing
- Filesystem - File operations

## Features
- Automated end-to-end testing workflows
- API testing and validation
- Database synchronization and verification
- Cross-browser testing
- Mobile app testing on real devices
- Professional Excel reporting

## Setup
```bash
pip install -r requirements.txt
```

## Running Tests
```bash
pytest tests/
```

## Architecture
This demo showcases integration between:
1. GitHub (version control and collaboration)
2. Azure DevOps (work item tracking and pipelines)
3. Sauce Labs (test execution platform)
4. MySQL (test data management)
5. Excel (test reporting)
